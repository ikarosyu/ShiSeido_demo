# 资生堂官方网上商城(https://www.shiseido.com.cn/)

### 首页 ：

- 登录/注册   php  cookie

- 基于百度开放接口的搜索(jsonp)

- 基于 Swiper 插件的轮播图

- 回到顶部

- 热销商品展示

- 抢购倒计时

  

### 列表页:

- 根据后台数据排序
- 点击商品携带id跳转详情页



### 详情页

- 基于ZOOM的放大镜

- 基于fly的购物车动画

- 配送区域的三级下拉菜单

- 输入框输入数量

- 限购功能

- 存localstorage

  

### 购物车

- 全选/单选  存localstorage
- 数量修改 （加 / 减 / 输入）修改localstorage
-  计算价格 
- 删除  修改localstorage
